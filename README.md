

👁️ Real-Time Face Detection System (OpenCV)

📌 Overview

This project is a real-time face detection system that uses a webcam to identify and highlight human faces instantly. It demonstrates the fundamental concepts of computer vision, image processing, and live video stream handling using Python and OpenCV.

The system captures video from the webcam, processes each frame, detects faces, and draws bounding boxes around them in real time.


🚀 Features

✅ Real-time face detection using webcam
✅ Detects multiple faces simultaneously
✅ Fast and lightweight implementation
✅ Works in different lighting conditions
✅ Visual bounding box around detected faces


🧠 Technologies Used

* 🐍 Python
* 👁️ OpenCV
* 📹 Webcam Video Capture
* 🤖 Haar Cascade Classifier


 ⚙️ How It Works

1. The webcam captures live video frames
2. Each frame is converted to grayscale
3. The Haar Cascade model scans for facial features
4. Detected faces are marked with rectangles
5. Output is displayed in real time
